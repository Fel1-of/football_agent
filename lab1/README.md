# Лабораторная работа 1 — Определение местоположения игрока

Реализация на Python 3 без внешних зависимостей.

## Что делает агент

- Подключается к `rcssserver` и отправляет начальную позицию `move`.
- После `play_on` вращается с заданной скоростью `turn`.
- По сообщениям `see` вычисляет координаты игрока по флагам.
- Выводит в консоль координаты игрока, мяча и противника (если видимы).

## Запуск

```bash
python3 app.py --team teamA --x -15 --y 0 --rotation 20
python3 app.py --team teamB --x 15 --y 0 --rotation 0
```

Перед запуском агентов поднимите `rcssserver` (и при необходимости `rcssmonitor`).
