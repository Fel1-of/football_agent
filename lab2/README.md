# Лабораторная работа 2 — Маршрут по флагам и удар по воротам

Реализация на Python 3 без внешних зависимостей.

## Что делает агент

- Подключается к `rcssserver` и после `init` определяет сторону (`side=l/r`).
- Выбирает маршрут по флагам зеркально стороне поля:
  - `side=l`: `fprb -> fgrb -> kick в gr`
  - `side=r`: `fplb -> fglb -> kick в gl`
- Движется к целевому флагу, затем подходит к мячу и бьёт в сторону видимых ворот.
- Если ворота не видны, выполняет запасной удар `kick 10 45`.
- После события `goal_*` маршрут начинается заново.

## Запуск

```bash
python3 app.py --team teamA --x -15 --y 0 --rotation 20
python3 app.py --team teamB --x 15 --y 0 --rotation 20
```

Перед запуском агентов поднимите `rcssserver` (и при необходимости `rcssmonitor`).
